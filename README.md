###How To Run Application 
1. Clone this repo
2. Run npm install
3. Run npm start
4. Goto localhost://3000
5. Click on the right most person icon on header to get login dialog
6. Give username in Lgin Dialog
7. Click on login
8. After login, Join the room using sidebar rooms
